Vad projektet gör

TMDB Movie Explorer är en webbsida som jag skapat där du kan:

Kolla på topp 10-betygsatta och mest populära filmer från TMDB.

Söka efter filmer och skådespelare.

Se filmens detaljer, som poster och beskrivning.

Skapa och hantera en egen watchlist som sparas i webbläsaren.
